# Qiita Project Group Index

> Qiita 記事を `#NN` 連載番号とは別に、project group 単位で辿るための索引。
> canonical な group 定義と運用ルールは [QIITA_POST_GUIDE.md](QIITA_POST_GUIDE.md) の `## 2. Project Group 単位でも束ねる` を正本とする。

## Scope

- この index の **正本スコープ**は `docs/articles/QIITA_#*.md` のうち `project_group` を持つ記事の全件である
- したがって **private article / companion / かみくだき版** も、`docs/articles/` 直下にある限り掲載対象に含む
- `docs/articles/drafts/` 配下は **補助アンカー**としてだけ載せる。主 group にまだ public article が無い場合、または関連導線の起点が必要な場合に限って明示する
- よって下の `gaitlab / spikelab / onocollo / llmesh` に見える draft 参照は、canonical scope そのものではなく **暫定の導線補助**である
