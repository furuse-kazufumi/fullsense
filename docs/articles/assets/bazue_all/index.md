> **出典**: 集英社『週刊ヤングジャンプ』公式 SNS 用素材ページ <https://youngjump.jp/info/bazue/>（SNS 等で無料使用可と明記）
> **作品**: 『スナックバス江』フォビドゥン澁川 / © Forbidden shibukawa / SHUEISHA
> **画像**: 本ディレクトリの `001.jpg`〜`206.jpg`（連番 206 枚・欠番なし）
> **文字起こし方法**: Claude（Opus 4.8）のビジョンで各画像を直接読解。縦書き・手書き風マンガフォントとも判読良好（手元の正解例 190.jpg と照合し一致を確認）。読み順はマンガ式（右→左・上→下）。

## 凡例
- 各エントリ = `### NNN.jpg` + **セリフ**（吹き出し / ナレーション枠 / 意味のある描き文字を読み順で）+ **場面**（誰・表情・状況の 1 行説明＝挿絵選びの手がかり）。
- **キャプション訳**: 各コマに en/zh/ko の 1 行要約キャプション（海外版記事の図キャプション用）。全セリフの逐語訳ではなくオチ/要点を凝縮。原作クレジット（© Forbidden shibukawa / SHUEISHA）は使用時に併記。
- ルビ（ふりがな）付きの語は一部 `親字（ルビ）` で併記。判読不能箇所は `(判読不能)`、セリフの無いコマは `(なし)`。
- 著作権透かし（©Forbidden shibukawa/SHUEISHA）・作品ロゴはセリフから除外。

## 注意点（文字起こし時に気づいたもの）
- **透かしロゴのパロディ表記**: 一部コマの作品ロゴが別作品名のパロディ（`パープル式部` / `パーチャル式部` 等。025/026, 028〜030, 040〜041 など）になっている。**これは『スナックバス江』自身のパロディ／メタ演出**（Wikipedia が作風として「パロディ・メタフィクション」を明記）であり、別作品の混入ではない。挿絵として問題なく使える。
- **074.jpg**: 吹き出しがロシア語「Свободное время」（＝ヒマ）にカタカナルビ。
- **156.jpg**: プロフィールの年収・身長が元画像で伏字。
- 描き文字（SFX）は意味のあるものをセリフ欄、状況描写は場面欄に振り分け。
