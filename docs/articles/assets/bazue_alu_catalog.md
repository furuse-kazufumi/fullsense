## これは何か

[Alu（アル, alu.jp）](https://alu.jp/) は、出版社公認でマンガの「コマ」（panel / crop）を共有・埋め込みできる正規サービス。各コマには **crop パーマリンク**が割り当てられており、この **permalink を埋め込むことで合法的に**記事へ掲載できる（生 `.jpg` を直リンクするのではなく、必ずパーマリンク経由で埋め込む）。

このカタログは、**『スナックバス江』（FORBIDDEN 渋川 / 集英社）** の Alu コマ一覧（series crops, page 1〜8）から、各コマの **permalink** と **キャプション（セリフ／シーン説明）** を機械抽出したもの。FullSense 系記事で「減点/加点方式」「避実撃虚」「honest disclosure」などのテーマに添えるリアクションコマを素早く選ぶための索引として使う。

- **抽出元**: `https://alu.jp/series/スナックバス江/crops?page=1` 〜 `?page=8`
- **抽出日**: 2026-06-15
- **注意**: 画像 `src` は静的 HTML に含まれないため未収録（合法な埋め込みは permalink で行うので問題なし）。キャプションは Alu 掲載テキストの OCR ベースで、誤字・脱字・伏字（〇）はそのまま転記している。空欄はページ上でキャプションが取得できなかったコマ（主に page 5・page 6 の一部、および page 4 後半）。
- **埋め込み方法**: 記事側では crop URL を Alu の埋め込み（oEmbed / 公式埋め込みコード）として貼る。直リンク画像は使わない。

> 凡例: キャプション空欄 = `—`。`(OCR乱れ)` はテキストが文字化けして判読不能だったコマ。
